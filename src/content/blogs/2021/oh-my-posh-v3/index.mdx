## Fundamental Differences Between V2 and V3
‚Ä¢ V2 was applicable for only Windows PowerShell. But V3 is available for Windows, macOS and Linux also.<br></br>
‚Ä¢ The configuration is changed from $ThemeSettings to .json to ensure easy customisation.<br></br>
‚Ä¢ More easy installation procedure.

## oh-my-posh V3: Step-by-Step Guide
#### Installation:
_I‚Äôll be using Windows PowerShell for this guide._ A PowerShell module has already been there for the effortless installation procedure. Just paste the following command in your PowerShell and press enter ‚Äî
```
Install-Module -Name oh-my-posh
```
Alternatively, you can use the installation command [Figure 1] given by the documentation itself (thereafter same procedure)‚Äî
```
Install-Module oh-my-posh -Scope CurrentUser
```

#### Setting Your Theme:
Now you can use the following command to set a theme as your terminal theme (microverse-power is a theme name provided by the oh-my-posh module itself)‚Äî
```
Set-PoshPrompt -Theme microverse-power
```
But wait ‚úã‚úã‚úã, it is not done yet. If you leave here, you have to set the theme again when you open the terminal next time. If you want to set the theme as permanent, i.e. when you open the terminal, the theme will be automatically set up for you, then you‚Äôll have to save the theme configuration in the profile. For that, copy-paste the following command in your terminal and hit enter ‚Äî
```
notepad $profile
```
It will open a notepad window. Copy-paste the above theme-setting command [Figure 5], i.e. ‚Äî
```
Set-PoshPrompt -Theme microverse-power
```

<div><div>

![Setting the theme as permanent](./figures/setting-the-theme-as-permanent.webp)

<sup>**Figure 5:** Setting the theme as permanent</sup>

</div></div>

Save the notepad file and run the command <span style="color:#da552e;">. $PROFILE</span> and finally reload your terminal to see the changes. At this point, you are good to go. Most people will be happy with all the available themes.

Getting an error: oh-my-posh command not found?

It‚Äôs because **oh-my-posh no longer support PowerShell Modules**. But don‚Äôt worry, they have given the solution. You just have to migrate oh-my-posh to enable it on your device. The process is very simple. Just copy-paste the following commands into your profile file (to use 1_shell)‚Äî
```
Import-Module oh-my-posh
oh-my-posh init pwsh --config https://raw.githubusercontent.com/JanDeDobbeleer/oh-my-posh/main/themes/1_shell.omp.json | Invoke-Expression
```
Now, if you want to use other themes, you can change the name of the theme in the link. You can get all the themes on their official website‚Äôs [theme section](https://ohmyposh.dev/docs/themes).

## Customization Procedure:
Basically, oh-my-posh renders your prompt based on the definition of _blocks_ (like Lego), which contain one or more _segments_. Now it‚Äôs more comfortable for you(!as a beginner) to modify one theme or merge two or more themes than to build from scratch. I have also combined two themes (**microverse-power** and **jandedobbeleer)**, which gave me the above result.

#### Blocks:
Under [**_General Settings_**](https://ohmyposh.dev/docs/configuration/overview), an essential part is _‚Äúblocks‚Äù_. Now under _‚Äúblocks‚Äù_, you can add the following elements ‚Äî<br></br>
‚Ä¢ type: <span style="color:#da552e;">prompt</span> ‚à£ <span style="color:#da552e;">rprompt</span> ‚à£ <span style="color:#da552e;">newline</span> <br></br>
‚Ä¢ alignment: <span style="color:#da552e;">left</span> ‚à£ <span style="color:#da552e;">right</span> <br></br>
‚Ä¢ vertical_offset: <span style="color:#da552e;">int</span> <br></br>
‚Ä¢ horizontal_offset: <span style="color:#da552e;">int</span> <br></br>
‚Ä¢ segments: <span style="color:#da552e;">array</span> of one or more <span style="color:#da552e;">segments</span> <br></br>

_**[1]. type:**_ Now, as you see, type can be of three types (I used prompt as type) ‚Äî
```yaml
ùêö) ùê©ùê´ùê®ùê¶ùê©ùê≠: renders one or more segments
ùêõ) ùê´ùê©ùê´ùê®ùê¶ùê©ùê≠: one or more segments aligned to the right of the cursor
ùêú) ùêßùêûùê∞ùê•ùê¢ùêßùêû: inserts a new line
```
_**[2]. alignment:**_ You can align the elements right or left using the alignment option (I used _left_ as alignment).

_**[3]. vertical_offset & horizontal_offset:**_ You can also give vertical and horizontal offset(space) using the vertical_offset and the horizontal_offset options, respectively (I didn‚Äôt use these parameters).

_**[4]. segments:**_ Now, segments are the most important part of this script. This part provides many parameters like ‚Äî

‚Ä¢ type: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ style: <span style="color:#da552e;">powerline</span> ‚à£ <span style="color:#da552e;">plain</span> ‚à£ <span style="color:#da552e;">diamond</span> <br></br>
‚Ä¢ powerline_symbol: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ invert_powerline: <span style="color:#da552e;">boolean</span> <br></br>
‚Ä¢ leading_diamond: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ trailing_diamond: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ foreground: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ foreground_templates: <span style="color:#da552e;">array</span> of <span style="color:#da552e;">string</span> values <br></br>
‚Ä¢ background: <span style="color:#da552e;">string</span> <br></br>
‚Ä¢ background_templates: <span style="color:#da552e;">array</span> of <span style="color:#da552e;">string</span> values <br></br>
‚Ä¢ properties: <span style="color:#da552e;">array</span> of <span style="color:#da552e;">Property</span>: <span style="color:#da552e;">string</span> <br></br>

Now I‚Äôm not going to discuss all the parameters. I‚Äôll talk about the most common and interesting ones ‚Äî
```yaml
ùü≠) ùòÅùòÜùóΩùó≤:  ùóºùòÄ-shows operating system you are using
          ùòÄùó≤ùòÄùòÄùó∂ùóºùóª-shows user name and host name
          ùóØùóÆùòÅùòÅùó≤ùóøùòÜ-shows battery status
          ùòÅùó∂ùó∫ùó≤-shows date and time
          ùóΩùóÆùòÅùóµ-shows the folder path
          ùòÄùóµùó≤ùóπùóπ-shows which type of shell you are using
          ùó¥ùó∂ùòÅ-shows git status (enabled with posh-git)
          ùó≤ùòÖùó∂ùòÅ-shows the display status (error/succesful)
ùüÆ) ùòÄùòÅùòÜùóπùó≤:  ùóΩùóºùòÑùó≤ùóøùóπùó∂ùóªùó≤-background(previous one)foreground(current one)
          ùóΩùóπùóÆùó∂ùóª-colored text on a transparent background
          ùó±ùó∂ùóÆùó∫ùóºùóªùó±-ifferent starting and ending symbol
ùüØ) ùóΩùóºùòÑùó≤ùóøùóπùó∂ùóªùó≤_ùòÄùòÜùó∫ùóØùóºùóπ: text character to use when style "powerline" is selected
ùü∞) ùó≥ùóºùóøùó≤ùó¥ùóøùóºùòÇùóªùó± ùóÆùóªùó± ùóØùóÆùó∞ùó∏ùó¥ùóøùóºùòÇùóªùó±: to set foreground and background colour 
ùü±) ùóΩùóøùóºùóΩùó≤ùóøùòÅùó∂ùó≤ùòÄ: to set different properties for different type like prefix, postfix, time format, display host, charging colour, discharging colour etc.
```
